# Movie Review Template Instructions

When provided with a **Movie Title**, generate a new markdown file named `[movie_title_in_snake_case].md` with the following structure.

Put all generated reference material inside a single HTML comment block `<!-- -->` at the top of the file to serve as a clean reference.

---

## Reference Material Structure (Inside Comments)

### 1. Detailed Plot Summary (Spoilers Included)
* A detailed, multi-paragraph breakdown of the entire plot from start to finish.
* Must include all major plot points, twists, and the ending in detail to help the writer recall the whole movie.

### 2. Chronological Scene Breakdown
* A chronological list of the movie's sequence of events.
* Provide roughly **5–10 bullets per hour** of the movie's runtime.
* Keep it brief but sequential (e.g., "1. Character arrives at X and witnesses Y", "2. Character discovers Z").

### 3. Themes & Analysis Topics
* A list of major themes, cinematic techniques, or artistic aspects commonly discussed in relation to this movie.
* **CRITICAL:** Do not include specific critics' opinions, reviews, or consensus to avoid bias. Only list the *topics* and *themes* people discuss.
